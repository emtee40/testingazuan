SpagoBI - iReport plugin
Before launching iReport, please porceed as follows:

1) copy file SpagoBIIReportPlugin-1.1.jar, present at current folder, to lib/ directory of iReport;

2) copy all jar files contained in lib/ directory (present at current folder) to lib/ directory of iReport;

3) copy file spagobi.xml to plugins/ directory of iReport;

4) In UX environment, copy iReport_SpagoBI_plugin_Resources directory with its content to bin/ directory of iReport.
     In WS environment, copy iReport_SpagoBI_plugin_Resources directory with its content under the iReport home directly.
