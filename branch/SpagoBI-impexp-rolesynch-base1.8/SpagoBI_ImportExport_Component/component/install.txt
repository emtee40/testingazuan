This component will allow you to export and import the SpagoBI business intelligence objects. 
To install the component do the following steps:

- go into the directory of the spagobi application of your application server 
  we will refer to that folder as SPAGOBI_APPLICATION_HOME
  (the directory contains some subdirectories 'jsp', 'css', WEB-INF, ....)
  
- copy the folders 'content' and 'WEB-INF' of this component into the SPAGOBI_APPLICATION_HOME 
  Some files will be overwritten and others they will be added
  
- Edit the file SPAGOBI_APPLICATION_HOME/WEB-INF/conf/master.xml and before of the last tag </MASTER> add the following lines
	
	<!-- start import export module -->
	<CONFIGURATOR path="/WEB-INF/conf/components/importexport/importexport.xml" />
	<CONFIGURATOR path="/WEB-INF/conf/components/importexport/pages.xml" />
	<CONFIGURATOR path="/WEB-INF/conf/components/importexport/modules.xml" />
	<CONFIGURATOR path="/WEB-INF/conf/components/importexport/presentation.xml" />
	<CONFIGURATOR path="/WEB-INF/conf/components/importexport/publishers.xml" />
	<!-- end import export module -->

- Edit the file SPAGOBI_APPLICATION_HOME/WEB-INF/web.xml, add the following lines after the other similar tags

	<servlet>
        <servlet-name>ExportServlet</servlet-name>
        <servlet-class>it.eng.spagobi.services.ExportServlet</servlet-class>
    </servlet>

	Then add the following lines after the other similar tags
	
	<servlet-mapping>
        <servlet-name>ExportServlet</servlet-name>
        <url-pattern>/ExportService</url-pattern>
    </servlet-mapping>

- Edit the file SPAGOBI_APPLICATION_HOME/WEB-INF/conf/spagobi/spagobi_components.xml and set to 'true' the 'installed'
  attribute of the tag <SPAGOBI_COMPONENT name="importexport" installed="true" /> 
  
- restart the server

- To use the import / export feature connect as admin and in the home page select the import / export link
