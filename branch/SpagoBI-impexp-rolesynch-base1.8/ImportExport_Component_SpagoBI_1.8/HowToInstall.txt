
This component will allow you to export and import the SpagoBI business intelligence objects. 

In order to install the component proceed as follows:

- go into the directory of the spagobi application of your application server; 
  we will refer to that folder as SPAGOBI_APPLICATION_HOME;
  the directory contains some subdirectories 'jsp', 'css', WEB-INF, ....

- copy all the folders of the component (components, css, jsp, WEB-INF) into the SPAGOBI_APPLICATION_HOME;
  Some files will be overwritten.
  
- go into the folder SPAGOBI_APPLICATION_HOME/WEB-INF/lib and delete the files spago-core-2.0.0.jar and spago-web-2.0.0.jar

- Edit the file SPAGOBI_APPLICATION_HOME/WEB-INF/web.xml and add the following lines after the other existing similar tags

    <servlet>
        <servlet-name>ExportServlet</servlet-name>
        <servlet-class>it.eng.spagobi.services.ExportServlet</servlet-class>
    </servlet>

  
  Then add the following lines after the other existing similar tags
	
    
    <servlet-mapping>
        <servlet-name>ExportServlet</servlet-name>
        <url-pattern>/ExportService</url-pattern>
    </servlet-mapping>

- restart the server

- To use the import / export feature connect as SpagoBI administrator and in the home page select the import / export link
