
This patch will enable the installation of SpagoBI modules on version 1.8. To install the patch do the following steps:

- go into the directory of the spagobi application of your application server 
  we will refer to the folder as SPAGOBI_APPLICATION_HOME
  (the directory contains some subdirectories 'jsp', 'css', WEB-INF, ....)
  
- copy the jsp and WEB-INF folders of this patch into the SPAGOBI_APPLICATION_HOME 
  Some files will be overwritten
  
- go into the folder SPAGOBI_APPLICATION_HOME/WEB-INF/lib and delete the files spago-core-2.0.0.jar and spago-web-2.0.2.jar
