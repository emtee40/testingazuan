SpagoBI - iReport plugin
The plugin has been tested with the iReport 1.2.6 version.

In order to install the plugin please copy the content of the directory SpagoBIIReportPlugin/plugin (contained into the downloaded zip file) into the root directory of your iReport program;
The content of the SpagoBIIReportPlugin/plugin folder has the same structure of the iReport root directory, so, some files will be overwritten.
After the copy operation start iReport and check that the plugins menu contains 'SpagoBI'.
