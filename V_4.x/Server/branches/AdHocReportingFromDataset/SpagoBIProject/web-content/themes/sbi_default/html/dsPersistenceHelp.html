Persistence of a dataset
In a datasets the user can make persisted its resulset.
The server will create a new table on the datasource selected with the structure of the resultet and full it
with the resultet content.
The name of the new table will be the label of the dataset managed. Is NOT possible to persist a dataset with parameters !!

Definition of a flat dataset
Is too possible to define a dataset as flat specifying its table name target.
If this option is checked the query of the dataset will be a generic "select * from < flat table name >"
