SpagoBI - iReport plugin
In order to compile the plugin, please compile SpagoBI Project and after proceed as follows:

1) move the current directory (SpagoBIIReportPlugin) into the root source directory of the SpagoBIPlatform (At the same level as the other projects QbeCore, SpagoBIProject, SpagoBIJPivotEngine, .....)

2) execute script file maven.xml (i.e. launch maven command);

3) after execution of maven command, please follows the instructions indicated in file "How to deploy the plugin.html" inside build/ folder.
